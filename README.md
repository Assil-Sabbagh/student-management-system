# Student Management System

A Python console application designed to help teachers manage student records efficiently.

## Features

* Add student name and score
* Display all students
* View summary (average, highest, lowest)
* Update student scores
* Delete students
* Search students (exact or partial match)
* Sort students (by name or score)
* Reset all data
* Automatic file saving (`students.txt`)

## Technologies Used

* Python
* Lists and Dictionaries
* File Handling
* Functions and Modular Programming

## How to Run

1. Download the project files
2. Open terminal in the project folder
3. Run:

```
python student_manager.py
```
